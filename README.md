# Mini Módulo de Faturamento com ACBr

Projeto desenvolvido em Delphi (VCL) simulando um módulo de faturamento ERP,
com geração de NF-e utilizando ACBr, controle de estados, eventos e logs.

## Tecnologias
- Delphi Alexandria
- VCL Forms
- Firebird 3
- FireDAC
- ACBr NF-e

## Funcionalidades
- Cadastro de clientes
- Cadastro de produtos
- Pedido de venda
- Geração de NF-e
- Simulação de retorno da SEFAZ
- Cancelamento de NF-e
- Histórico de eventos e logs
